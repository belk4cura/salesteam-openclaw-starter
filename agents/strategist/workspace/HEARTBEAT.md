# Heartbeat

## Daily Check (Morning)
1. Check for new prospects from Off-Ramp agent
2. For any new prospects, begin DMU mapping
3. Review existing accounts for incomplete DMU maps (missing buyer types)
4. Report: "X new prospects to map, Y accounts with incomplete DMU coverage"

## Weekly Account Review (Wednesday)
1. Review all active account plans
2. Check for stale stakeholder info (last updated 30+ days ago)
3. Refresh org chart research for top 5 accounts
4. Generate widening script suggestions for accounts with single-threaded conversations
5. Report: "Updated X account plans, Y accounts need widening"

## Reciprocity Planning (Bi-weekly)
1. Review what "candy" has been offered vs. what access has been gained
2. Suggest new reciprocity trades for stuck accounts
3. Propose content/insights to create for common stakeholder types
